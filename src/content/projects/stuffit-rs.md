StuffIt-RS is a Rust project for working with StuffIt archives. It sits close
to the preservation work because file formats are often where old software
becomes unreachable first.

The useful part of a format tool is not only speed. It is the confidence that a
file can be inspected, moved, extracted, and rebuilt without losing the meaning
stored inside it.
