Systemless is an experiment in preserving old software as a living interface.
The goal is not to make the past look modern. It is to give old software enough
of its surrounding world that it can be understood on its own terms.

<figure>
  <img src="/images/figures/archival-interface.png" alt="An archival figure of a preserved desktop software environment" />
  <figcaption>
    Figure 1. Old software is easier to study when its interface is treated as
    an artifact rather than a skin.
  </figcaption>
</figure>

The project is built around a simple idea: execution is only part of
preservation. The file system, display model, input rhythms, and windowing
assumptions all belong to the object being preserved.

Future notes on Systemless will live here as the project becomes more concrete.
