m68k-rs is a pure Rust implementation of the Motorola 68000 family of CPUs.
For preservation work, CPU behavior is not an abstract detail. It is the floor
that the rest of the old system expects to stand on.

Projects like this are slow, exacting, and useful because they make old
software less dependent on old hardware.
